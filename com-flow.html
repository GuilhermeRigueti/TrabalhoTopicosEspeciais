Funcionalidades

git flow feature start NOME: Comando para iniciar o desenvolvimento de uma nova funcionalidade

git flow feature finish NOME: Comando que mesca o NOME no develop, remove o branch da funcionalidade e volta para o branch develop

git flow feature publish NOME: comando para publicar a funcionalidade no servidor remoto

git flow feature pull NOME: Comando para obter uma funcionalidade publicada por outro usuário

Criar versão / Release

git flow release start RELEASE [BASE]: Comando para começar uma nova versão, cria um branch da versão baseado no bransh develop

git flow release publish RELEASE: Comando para publicar o branch da versão após sua criação, permitindo os commits de outros desenvolvedores

git flow release track RELEASE: Comando para acompanhar versão remota

Finalizar Versão

git flow release finish RELEASE: Comando que executa ações como mesclar o bransh da versão master, etiqueta a versão com seu nome, mescla o branch da versão de volta no develop, remove o branch da versão

Hotfix

git flow hotfix start VERSION [BASENAME]: Comando para iniciar um hotfix

git flow hotfix finish VERSION: Comando para finalizar um hotfix
