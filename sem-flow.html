Git Add

Git add .: O comando add é usado para adicionar qualquer alteração de arquivo ao INDEX do git

Git commit

Git commit -a -m "(mensagem do commit)", O Commit é utilizado para adicionar as alterações ao histórico do repositório atribuindo um nome ao commit

Git push

O comando git Push é usado para fazer o envio do conteúdo que está presente no repositório local para um repositório remoto

Git branch

O Branch nada mais é do que uma ramificação que no git se refere a um ponteiro para as alterações que são realizadas nos arquivos do projeto
